\hyperlink{class_c3_p_o_network}{C3\-P\-O\-Network} is a class facilitating network communication for the C3\-P\-O project. Only basic networking is managed here.

Basic operations are\-:
\begin{DoxyItemize}
\item Create and connecting to the teacher server (which also holds a Unity Master\-Server)
\item Create and connecting to game servers for multi-\/player
\end{DoxyItemize}

This class is built against the singleton model
